# Architecture Overview

This application simulates an internal Enterprise IT admin platform used by multiple subsidiaries.

## Core Concepts
- Multi-tenant application
- Each tenant represents a distinct legal entity or subsidiary
- Users belong to one tenant
- Roles include employee, manager, it_admin, and compliance_admin

## Features
- Authentication
- Employee profile lookup
- Support tickets
- Evidence uploads
- Approval workflows
- Audit export
- URL import for administrative workflows
